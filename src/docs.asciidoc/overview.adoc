[[overview-http-status-codes]]
=== HTTP status codes

|===
| 상태 코드 | 설명

| `200 OK`
| 요청 성공

| `201 CREATED`
| 새로운 리소스 생성

| `204 NO CONTENT`
| 요청 성공 및 반환 컨텐츠 없음

| `400 Bad Request`
| 잘못된 요청

| `401 Unauthorized`
| 비인증 상태

| `403 Forbidden`
| 권한 거부

| `404 Not Found`
| 존재하지 않는 리소스 요청

| `500 Internal Server Error`
| 서버 내부 에러
|===
